title: Review: Google's Deep Learning Course on Udacity
---

https://www.udacity.com/course/deep-learning--ud730

- topics: intro to neural networks (NNs), logistics classifiers and gradient descent; deep NNs; regularization; convolutional NNs; embeddings; long short-term memory recurrent NNs
- split into 4 lessons with 6 assignments; application of deep NNs using TensorFlow
- good intro to TensorFlow
- great that code on GitHub and assignments in Jupyter Notebook
- quite shallow treatment of topics
- provides some pointers to literature so people interested in a specific topic can dig deeper
- great style of teaching (Vincent Vanhoucke)
- interesting problems in assignments but not a lot of effort was put into specifying tasks clearly which makes the problems more complicated than they would need to be
-> forum helps; overall good assignments
